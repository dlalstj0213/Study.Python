# 자료형

### 자료형 : Data Type

- 자료형이란 자료의 형식을 뜻하는데, 이러헌 형식은 자료의 기능과 역할에 따라 구분되어있다.

## 가본 자료형

- 문자열 (string)
- 숫자 (number)
- 불, 참/거짓 (boolean)

## `type()` : 자료 형식 확인 함수

```python
print(type("문자입니다."))
# 결과 : 
# <class 'str'>
print(type(1234))
# 결과 : 
# <class 'int'>
print(type(true))
# 결과 : 
# <class 'bool'>
```
